Pasos para trabajar con GIT Local

  1. Iniciar un repositorio local: git init
  2. Agregar archivos al área de preparación (todos): git add .
  3. Agregar archivos al área de preparación: git add archivo
  4. Confirmar los cambios: git commit -m "Mensaje del commit"
